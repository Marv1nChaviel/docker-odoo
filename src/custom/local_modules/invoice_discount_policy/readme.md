## Descripción
Este módulo automatiza la aplicación de descuentos en facturas de venta basándose en políticas predefinidas vinculadas al tipo de cliente o categoría.

## Características Principales
- **Reglas de Descuento**: Configure porcentajes de descuento globales o específicos.
- **Tipos de Cliente**: Asocie descuentos a etiquetas o tipos de partner específicos.
- **Aplicación Automática**: El descuento se calcula y aplica automáticamente al crear o validar la factura.

## Uso
1. Configure las Políticas de Descuento en Contabilidad > Configuración.
2. Asigne la categoría o tipo correspondiente al Cliente.
3. Al crear una factura para ese cliente, el descuento se aplicará automáticamente.
