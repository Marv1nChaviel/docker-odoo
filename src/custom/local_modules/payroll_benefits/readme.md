## Descripción
Módulo para la gestión y cálculo automatizado de beneficios laborales en la nómina, soportando configuraciones complejas y múltiples tipos de contrato.

## Características Principales
- **Configuración Flexible**: Defina reglas para diferentes tipos de beneficios (ej. bonos, primas).
- **Integración con Nómina**: Se integra directamente con el motor de cálculo de nómina de Odoo.
- **Soporte Multi-contrato**: Permite configuraciones específicas según el tipo de contrato del empleado.
- **Seguridad**: Reglas de acceso para configurar quién puede ver o editar beneficios.

## Uso
1. Vaya a Nómina > Configuración > Beneficios.
2. Defina los parámetros y reglas de cálculo.
3. Los beneficios se procesarán automáticamente al generar las nóminas de los empleados elegibles.

## Dependencias
- `hr_payroll`
- `hr_contract`
