## Descripción
Este módulo introduce un sistema de etiquetado dinámico para productos en el almacén, facilitando la organización visual y lógica del inventario.

## Características Principales
- **Gestión de Etiquetas**: Cree etiquetas con nombres y colores personalizados.
- **Asignación Rápida**: Asigne etiquetas a productos desde la vista de lista o formulario.
- **Vistas Agrupadas**: Visualice el inventario agrupado por etiquetas de almacenamiento en vista Kanban y Lista.
- **Wizard de Asignación Masiva**: Herramienta para asignar etiquetas a múltiples productos simultáneamente.

## Uso
1. Vaya a Inventario > Configuración > Etiquetas de Almacenamiento.
2. Cree las etiquetas necesarias (ej. "Refrigerado", "Zona A", "Frágil").
3. Asigne las etiquetas a los productos.
4. Use los filtros y agrupaciones en la vista de Inventario para organizar su stock.

## Dependencias
- `stock`
- `product`
